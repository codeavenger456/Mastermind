Mastermind is a game of guessing the secret combination

In this game, secret combination is restricted to have 4 non-repeating digits

It has Mandarin version and English version

Detailed thought process is in the comments of mastermind.m file

However, there is a line limit in matlab editor which cannot see the entire program

To see the whole program, download the file

To deploy or to see the UI of the game, please see mastermind.mlapp file
